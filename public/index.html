My API - Movies and Actors

How to get all Movies

To get all available movies type this in the browser: "http://localhost:5000/api/movies"

How to get a specific movie

To get a specific movie type this in the browser: "http://localhost:5000/api/movies/:movieID" (Change :movieID to the actual id of the movie you want to see)

How to get all Actors

To get all available actors type this in the browser: "http://localhost:5000/api/actors"

How to get a specific actor

To get a specific actor type this in the browser: "http://localhost:5000/api/actors/:actorID" (Change :actorID to the actual id of the actor you want to see)
